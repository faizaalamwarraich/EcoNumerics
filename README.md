# EcoNumerics-
River Pollution Load Modeling &amp; Environmental Analytics Platform (Python, Streamlit, Pandas, NumPy, Plotly, ReportLab, Matplotlib)
- Built a data analytics web application in Python using Streamlit for the reactive UI layer, with Pandas and NumPy powering all backend data processing and numerical computation in a single integrated app.
- Implemented custom numerical integration algorithms in NumPy (Trapezoidal, Simpson's 1/3 & 3/8, Boole's, and Weddle's rules) to model pollutant mass loading, with a Pandas-driven cross-method accuracy deviation matrix rendered via conditional styling.
- Designed an ETL data pipeline with Pandas and openpyxl to ingest and normalize multi-sheet Excel datasets across 22 chemical parameters and two seasonal datasets, leveraging Streamlit's @st.cache_data for optimized reload performance.
- Developed interactive visualizations with Plotly (dark-themed, annotated scatter/line charts with hotspot markers) and automated report generation using ReportLab and Matplotlib to export analysis results as downloadable PDFs.
